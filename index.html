Projects

CartShare

Tech Stack - React, OAuth2, Spring Boot, JPA, MySQL, Docker

Cartshare is a grocery shopping application which allows the shoppers to form a pool to reduce their outings to the grocery stores in the time of COVID-19. Frontend is developed in React, Redux and the backend is made up of multiple Spring Boot concepts like AOP, JMS, security, JPA, Persistence & Transaction. The Docker containers have been deployed on multiple ec2 instances. S3 bucket is used to store images and Amazon RDS for Database.

AISight

Tech Stack - RaspberryPi, NLP, Image Captioning

AIsight is a portable device (RaspberryPi) which with the help of Computer Vision and Natural Language Processing helps visually impaired people see the world.
